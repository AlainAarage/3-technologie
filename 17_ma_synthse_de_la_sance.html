1.7 Ma synthèse de la séance

en ilot

Répondez aux question ci dessous

-Donner le nom du 1er mode de fonctionnement du logiciel "Filius".

-Quelle est l'utilité de ce mode?

-Donner le 2ème mode de fonctionnement du logiciel "Filius".

-Quelle est l'utilité de ce mode?

-Donner le nom de la commande utilisée pour vérifier l’adresse IP d’une machine?

-Donner le nom de la commande utilisée pour vérifier la communication entre 2 machines?

-Quel type de logiciel est il nécessaire d'installer sur Filius pour utiliser les 2 commandes précédentes?

-Quelle est le nom de la commande permettant d'utiliser ces 2 commandes sur un vrai ordinateur?

-Que permettent les commutateurs dans un réseau?

-Quelles est le rôle d'une "passerelle" dans une configuration IP d'une machine?

-Deux machines d’un même réseau peuvent-elles communiquer sans passerelle? (justifier votre réponse) 

-Deux machines d’un réseau différents peuvent-elles communiquer sans passerelle? (justifier votre réponse)

-A l’aide de l’invite de commande (cliquer sur l’icone Windows  puis taper CMD), donner l’adresse IP de votre ordinateur, puis l’adresse IP de sa passerelle (son serveur).   

-Pinguer sur votre serveur afin de voir si vous pouvez l’atteindre.

-Réaliser une capture d’écran de vos 2 résultats précédents (Adresse IP + ping).
